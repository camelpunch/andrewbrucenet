# andrewbruce.net

This is the source for www.andrewbruce.net

## Installation

`git clone` it.

Install [Leiningen](https://github.com/technomancy/leiningen).

## Development

`lein ring server`

## Deploy to Cloud Foundry

```
lein uberjar
cf push
```
